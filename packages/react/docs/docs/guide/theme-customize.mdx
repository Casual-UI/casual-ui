### Theme colors

```jsx live
function Demo() {
  const colors = [
    '--casual-primary',
    '--casual-secondary',
    '--casual-warning',
    '--casual-negative',
  ]

  return (
    <div className="c-row c-gutter-md c-items-center c-wrap">
      {colors.map(colorName => (
        <div className="c-col-3" key={colorName}>
          <div
            className="c-rounded-md color-card c-pa-md c-flex c-items-center c-justify-center"
            style={{
              backgroundColor: `var(${colorName})`,
              height: '120px',
              lineHeight: '120px',
              color: '#fff',
              textAlign: 'center',
            }}
          >
            {colorName}
          </div>
        </div>
      ))}
    </div>
  )
}
```
:::tip TIP
All of the colors used in Casual UI based on [CSS Variables](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties).

You can override them in your custom css.

:::


### Variables Definition
