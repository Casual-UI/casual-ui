### 安装依赖

```shell
npm install casual-ui-react react react-dom
```

:::tip 注意
需要单独安装`react`以及`react-dom`
:::

### 使用

- 入口文件引入样式

```js
import 'casual-ui-react/style.css'
```

- JSX 中使用

```jsx
import { CButton } from 'casual-ui-react'

const APP = () => <CButton label="按钮" />
```
