## Packages Manage

- global install pnpm

```sh
npm install -g pnpm
```

- use pnpm to install all dependencies

```sh
pnpm install
```

## IDE Settings

Use VSCode and extension [Volar](https://github.com/johnsoncodehk/volar). And open the [Take over mode](https://vuejs.org/guide/typescript/overview.html#takeover-mode)
